## Сборка

Запуск сборки проекта.

```sh
npm run build
```

## Тестирование

Запуск unit-тестов

```sh
npm test
```

Запускаем сервер в папке `js-advanced-diploma`
```sh
npm start
```
Затем сборку проекта
```sh
npm run dev
```

После cypress для e2e тестов

```sh
npx cypress open
```

После открытия заустить JS файл `cypress/e2e/basic-functionality.test.js`
